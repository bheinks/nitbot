# nitbot
```
N - Not
I - In
T - The
B - Bible
O
T
```

A simple Discord bot that, when mentioned either directly or in reply to a message, can tell you what words are/aren't included in the Bible.

## Requirements
```
python = "^3.10"
discord-py = "^2.2.2"
python-decouple = "^3.8"
nltk = "^3.8.1"
contractions = "^0.1.73"
```

## Installation
```
pip install nitbot
```
